# capitole-inditex
Api rest to search prices.

## Technologies:  

* **Java 17**
* **Maven**
* **Spring Boot**
* **Spring Security**
* **JUnit**
* **Swagger**
* **H2 Database**

## Use:

This service needs to set the next enviroment variables

Database:

 - database_user = username.
 - database_pass = password.
 
Spring security:

 - security_user = username.
 - security_pass = password.
